# Web Frontend

React + TypeScript frontend for Axiom IDP.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

- `src/components/` - React components
- `src/pages/` - Page components
- `src/lib/` - Utilities and API client
- `src/store/` - Zustand state management
- `src/styles/` - Tailwind CSS styles
- `public/` - Static assets

## Testing

```bash
npm test
npm run test:ui
```

## Linting

```bash
npm run lint
npm run format
```
